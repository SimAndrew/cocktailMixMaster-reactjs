## Cocktail Mix Master

### About the Project

A React JS web application that allows users to explore a wide variety of cocktail recipes. Users can search for
cocktails by name or ingredient, view detailed recipes, and discover new drinks to try. The app fetches data from
TheCocktailDB API and provides an intuitive interface for cocktail enthusiasts.

### Technologies, Libraries:

- [React JS](https://react.dev/)

- [Vite JS](https://vitejs.dev/), [ESLint](https://eslint.org/), [Prettier](https://prettier.io/)

- https://www.thecocktaildb.com/api.php TheCocktailDB is a free and open database of cocktail recipes and drink
  information, providing an API for developers to access a wide variety of cocktail data, including ingredients,
  instructions, and images.

- https://axios-http.com/docs/intro Axios is a popular, promise-based JavaScript library for making HTTP requests from
  browsers or Node.js, simplifying tasks like fetching data from APIs with methods like GET, POST, and DELETE. It offers
  features like interceptors, automatic JSON transformation, and better error handling, making asynchronous network
  operations cleaner and more efficient than native browser APIs.

- https://reactrouter.com/home React Router is a standard library for routing in React applications, enabling developers
  to create single-page applications with navigation capabilities by mapping URL paths to specific components, allowing
  for dynamic rendering of content based on the current route.

- https://styled-components.com/docs Styled-components is a popular CSS-in-JS library for React that lets developers
  write actual CSS syntax directly in their JavaScript files, creating component-scoped styles using tagged template
  literals, which enhances component organization, avoids naming conflicts, and enables dynamic styling based on props.

- https://github.com/fkhadra/react-toastify React-Toastify is a popular React library that allows developers to easily
  add customizable toast notifications to their applications, enhancing user experience by providing timely feedback and
  alerts in a visually appealing manner.

- https://tanstack.com/query/latest/docs/framework/react/overview React Query is a powerful data-fetching library for
  React applications that simplifies the management of server state by providing tools for fetching, caching,
  synchronizing, and updating data, allowing developers to build more efficient and responsive user interfaces with less
  boilerplate code.
